Progress check

Now would be a good time to do a progress check because we did a lot of work.

Before going any further you should have:

  • A DynamoDB table named YOUR_FIRST_NAME-savealife-dev with PK being the primary key
  • “Donor sign-up” AWS Lambda function deployed and persisting the data to the above-mentioned table
  • “Donation event” AWS Lambda function deployed and persisting the data to the same table

If any of these is not working for you - please let me know. If everything mentioned above worked for you, great!
